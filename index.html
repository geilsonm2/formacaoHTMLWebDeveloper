Falando sobre Tags e seus atributos

Existem atributos globais para todas as tags e outros que são mais específicos para cada Tag.

Exemplo de tag global:

Strong

A tag Strong como exemplo pode ser atribuido a ela um id, uma class entre outro atributos. Com um id, o JavaScript consegue acessa-la com facilidade.
Passando o atributo style, conseguimos alterar a aparencia daquela tag utilizando o CSS.

Exemplo de tag especificas:

  • Input
  • Img
  • Hr

Textos

Tags de Personalização

  • Strong = Negrito
  • u = Sublinhar
  • Mark = Marcação estilo marca-texto
  • i = Itálico
  • sup = Dessa forma2

Tags de Exibição de Textos

  • h1 = Títulos
  • h2 = Sub-Títulos
  • p = Paragrafo
  • blockquote = Citações
  • small = Textos menores, dessa forma: small

Listas Ordenadas e Não Ordenadas

Listas Ordenadas são ordenadas por números a cada item da lista.

Exemplo de Listas Ordenadas

Utilizamos ol e dentro do ol colocamos li list-item
  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de Listas Não Ordenadas

Utilizamos ul e dentro do ul também colocamos li list-item
  • Item 1
  • Item 2
  • Item 3

Links

Utilizamos a tag "a" (ancor) para atribuir uma pagina aquele link

Exemplos de atributos da tag "a":
  • href = Para indicarmos a página que sera redirecionada
  • title = Para descrevermos onde seremos direcionados
  • target = Na tag target utilizamos _self para abrir a nova pagina na mesma pagina que estamos
  • target = Na tag target também podemos utilizar _blank para abri uma nova guia.
    • Dentro do href também conseguimos passar um id, orientando para qual parte deve ser redirecionado dentro da mesma pagina!
